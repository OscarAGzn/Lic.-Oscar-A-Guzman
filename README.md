Certificación de PowerBI
## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un dashboard interactivo en Microsoft Power BI enfocado en el análisis de órdenes de compra, compradores y ventas mundiales. El objetivo principal fue transformar una base de datos proporcionada con fines académicos en una herramienta visual capaz de facilitar la interpretación de información comercial mediante gráficos dinámicos, mapas y comparaciones anuales.

El dashboard fue diseñado para mejorar la visualización y optimización de datos, permitiendo presentar información de manera profesional y comprensible para la toma de decisiones y exposición de resultados.

Procedimiento Realizado

### 1. Importación de Datos

Se trabajó con una base de datos proporcionada por el profesor, la cual fue importada a Power BI para su análisis y procesamiento.

### 2. Transformación y Preparación de Datos

Durante la etapa de limpieza y transformación se realizaron las siguientes actividades:

Creación de columnas calculadas
Organización y estructuración de datos
Preparación de información para análisis comparativos
Relación de variables relacionadas con compradores y ventas

Estas transformaciones permitieron obtener una base más clara y funcional para la construcción del dashboard.

### 3. Desarrollo de Visualizaciones

Se desarrollaron diferentes elementos visuales con el objetivo de representar la información de manera clara e interactiva:

Gráficas comparativas de ventas
Comparaciones de ventas por año
Mapas de ventas mundiales
Indicadores visuales para facilitar el análisis de datos

Las visualizaciones fueron diseñadas para mejorar la interpretación de tendencias comerciales y distribución geográfica de ventas.

### 4. Análisis de Datos

El análisis principal se centró en:

Comparación de ventas entre distintos años
Identificación de comportamiento de compradores
Visualización del alcance mundial de ventas
Interpretación de patrones comerciales mediante gráficos dinámicos
### 5. Objetivo del Dashboard

El dashboard tiene como finalidad optimizar la visualización de información y facilitar el entendimiento de grandes volúmenes de datos comerciales mediante herramientas visuales interactivas y profesionales.
